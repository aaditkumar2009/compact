### Students will be able to:

- Be versed with the technical terms like WWW, Web browser, Web server, Website, Web page, HTML, Web address, and TCP/IP
- Discuss various Internet protocols in the class

#### At the end of the chapter, students will be able to:

- Be thoroughly versed with Internet-related terminologies and the set of protocols to be followed while using Internet services.
- Answer questions correctly.
